# Getting Started with XSpace

XSpace is a chain-agnostic website builder and CMS for Web3 communities, by the creators of [XBallot](https://xballot.net) and [Pipeline-UI](https://www.pipeline-ui.com) at [HEADLINE](https://github.com/headline-design).

XSpace, the revolutionary Web3-native website builder, streamlines app development with its chain-agnostic engine and cutting-edge CMS. Here's a quick guide to get you started on your journey with XSpace.

## Key Features

- **Drag-and-Drop Interface:** An intuitive no-code interface simplifies the app development process.
- **Cross-Chain Compatibility:** XSpace provides out-of-the-box support for blockchain L1s, L2s, and beyond.
- **Lightning-Fast Rendering:** XSpace's native hot-reload feature offers a novel visual element to Web3 development.
- **P2P Data Storage:** XSpace uses enterprise-grade P2P storage solutions for seamless content retrieval.
- **Multi-Environment Optimization:** Build multi-tier decentralization concurrency directly into your website.
- **Complete Interoperability:** Build websites and Dapps that work seamlessly across legacy L1s and the latest chains.
